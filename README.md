# grdw.nl

My personal blog.

I'm currently in the process of migrating this blog away from GitHub pages, and setting it up in such a way that it can run from the tiniest of hardware.

## Setup

### For the Raspberry Pi B+
1. Download [alpine](https://alpinelinux.org/downloads/)
1. Flash an SD card to FAT32
1. Extract the contents from the `.tar.gz` onto the SD-card
1. ...
1. Profit
